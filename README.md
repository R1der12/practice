# Practice

Проект, выполненный в рамках производственной практики.

Цель работы — автоматизация развертывания веб-приложения с использованием инструментов контейнеризации и управления конфигурацией.

## Stack

* Python / Flask
* PostgreSQL 16
* Nginx
* Docker
* Docker Compose
* Ansible

## Overview

Проект представляет собой веб-приложение, развертываемое в контейнеризированной среде. Для автоматизации настройки сервера и деплоя используется Ansible.

Инфраструктура включает:

* веб-приложение на Flask;
* базу данных PostgreSQL;
* Nginx в качестве reverse proxy;
* Docker Compose для оркестрации контейнеров;
* Ansible для автоматизированного развертывания.

## Features

* Контейнеризация приложения
* Reverse Proxy через Nginx
* Поддержка HTTPS
* Хранение данных в PostgreSQL
* Автоматизированное развертывание через Ansible
* Управление сервисами через Docker Compose

## Deployment

### Docker Compose

```bash
docker compose up -d
```

### Ansible

```bash
ansible-playbook playbook.yml
```

> Playbook выполняет подготовку сервера, установку необходимых зависимостей и развертывание приложения.
